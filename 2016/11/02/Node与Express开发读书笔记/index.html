初识Express

Fast, unopinionated, minimalist web framework for Node.js

Express特性

  • 精简
    Express的哲学在于充当你的想法和服务器之间薄薄的一层，同时拥有足够的健壮性
  • 灵活
    可扩展，需要的时候再添加
  • Web程序框架
  • 单页Web程序
    SPA将整个网站或其中很大一部分都下载到客户端浏览器上，经过初始下载后，用户访问不同页面的速度会加快，因为减少了大部分HTTP请求
  • 多页和混合Web程序

静态资源服务

用Node提供静态资源只适用于初期的小型项目，对于较大的项目，应该使用Nginx或CDN之类的代理服务器来提供静态资源

省时省力的Express

app.METHOD(path,callback)``app 是 express 对象的一个实例， METHOD 是一个 HTTP 请求方法， path 是服务器上的路径， callback 是当路由匹配时要执行的函数。
app.use([path,]function) 是Express添加中间件的一种方法

在定义路由中的路径时，app.METHOD为我们做了很多中间工作:默认忽略大小写及反斜杠，同时在进行匹配时不考虑查询字符串
另外,路由和中间件的添加顺序非常重要
