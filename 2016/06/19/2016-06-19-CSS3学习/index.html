CSS3学习

Posted by Rubicker on 2016-06-19

掌握CSS3中新特性

属性选择器

E[attr]
E[attr=value]
E[attr^=value] attr属性以value值开头的元素
E[attr$=value] attr属性以value值结尾的元素
E[attr*=value] attr属性中有value值的元素
E[attr~=value] attr属性中有value词的元素，注意此时属性值为一个词列表
E[attr|=value] attr属性中以value或者value-开头的值

结构性伪类

E:nth-child(x) E标签父级下第x个子元素，且该元素若不是E元素则不起作用。x从1开始；x可取odd，even，2n-1类似的算法(n从0开始算)
E:nth-last-child(x) 从最后一个算起，用法同上
E:nth-of-type(x) E标签父级下第x个E元素，只匹配E元素
E:nth-last-of-type(x) 从后算起

E:first-child == E:nth-child(1)
E:last-child == E:nth-last-child(1)
E:first-of-type == E:nth-of-type(1)
E:last-of-type == E:nth-last-of-type(1)

E:empty 选取无子节点的E标签。注意：文本节点也算子节点

E:only-of-type 选取父元素只有唯一一个E标签的E标签。注意：子节点不包含文本节点

伪类

E:target 表示当前URL片段的元素类型。这个元素必须是E。匹配相关URL指向的E元素，即哈希值（#后面的）
E:disabled 表示不可点击的表单控件
E:enabled 表示可点击的表单控件
E:checked 表示已选中的checkbox或radio
E:first-line 表示E元素中的第一行
E:first-letter 表示E元素中的第一个字符
E::selection 表示E元素在用户选中文字时
E::before 生成内容在E元素前
E::after 生成内容在E元素后 content属性
E:not(s) 表示E元素中不包含s的元素
E~F 表示E元素之后的所有F元素

新增文本属性

新增颜色模式

rgba ——a指alpha，透明度0-1

文字阴影

text-shadow
阴影可以叠加，每层用逗号隔开。

文字描边

-webkit-text-stroke:宽度 颜色

新增文本功能

Direction 定义文字排列方式（全兼容）

  • Rtl 从右向左排列
  • Ltr 从左向右排列
  • 配合unicode-bidi一块使用 同时设置unicode-bidi:bidi-override才可生效

Text-overflow定义省略文本的处理方式

  • clip 无省略号
  • ellipsis 有省略号(配合overflow:hidden和white-space:nowrap(强制不换行)一起使用)

@font-face

自定义文本字体

弹性盒模型

  • 注意在使用弹性盒模型的时候，父元素必须要加display:box或display:inline-box
  • box-orient定义盒模型的布局方向
    • horizontal 水平显示
    • vertical 垂直显示
  • box-direction 元素排列顺序
    • normal 正序
    • reverse 反序
  • box-ordinal-group 设置元素的具体位置
  • box-flex 定义盒子的弹性空间
    子元素的尺寸=盒子的尺寸*（子元素的box-flex属性值/所有子元素的box-flex属性值的和）
  • box-pack对盒子富余的空间进行管理
    • start 所有子元素在盒子左侧显示，富余空间在右侧
    • end 所有子元素在盒子右侧显示，富余空间在左侧
    • center 所有子元素居中
    • justify 富余空间在子元素之间平均分布
  • box-align 在垂直方向上对元素的位置进行管理
    • start 所有子元素在最顶
    • end 所有子元素在最底
    • center 所有子元素居中

盒模型阴影

  • box-shadow:[inset] x y blur [spread] color
    • inset:投影方式为内投影，不给则为外投影
    • x,y 阴影偏移量
    • blur 模糊半径
    • spread 扩展阴影半径 (先扩展原有形状，再开始画阴影)
    • color 阴影颜色

其他新增属性

  • box-relect 倒影
    • direction 方向 above|below|left|right
    • 距离
    • 渐变(可选)
  • resize 自由缩放，可拖拽
    • both 水平垂直都可以缩放
    • horizontal只有水平方向可以缩放
    • vertical 只有垂直方向可以缩放
    • 注意：一定要配合overflow:auto一块使用
  • box-sizing 盒模型解析模式
    • content-box 标准盒模型
    • border-box 怪异盒模型

CSS3响应式布局

分栏布局

  • column-width 栏目宽度
  • column-count 栏目列数
  • column-gap 栏目距离
  • column-rule 栏目间隔线

媒体查询

CSS3新增UI

border-radius

可以分别设置x，y轴半径，用/分隔
各种长度都可以：px,%

边框

边框图片 border-image

  • border-image-source 引入图片
  • border-image-slice 切割图片 数字 但是不写px
  • border-image-width 边框宽度
  • border-image-repeat 图片排列方式 round 平铺 repeat stretch

边框颜色 border-colors

多层颜色 用空格间隔

线性渐变

  • linear-gradient([<起点>||<角度>]？<点>，<点>…)
  • 只能用在background上
    • IE
      filter:progid:DXImageTransform.Microsoft.gradient(startColorstr=”#ffffff”,endColorstr=”#ff0000”,GradientType=”1”)
  • 参数
    • 起点：从什么方向开始渐变 默认为top
      • left，top，left top
    • 角度：从什么角度开始渐变
      • xxx deg的形式
    • 点：渐变点的颜色和位置
      • black 50% 位置可选
    • 平铺
      • 默认的background-repeat：repeat对渐变不起作用；
      • 可以使用repeating-linear-gradient 进行渐变平铺

径向渐变

  • radial-gradient([<起点>]?[<形状>||<大小>]?<点>,<点>)
    • 起点，可以是关键字(left top right bottom),具体数值或百分比
    • 形状，ellipse,circle
    • 大小，具体数值或百分比，也可为关键字(closest-side,closest-corner,farthest-side,farthest-corner,contain or cover)
      • firefox 目前只支持关键字
