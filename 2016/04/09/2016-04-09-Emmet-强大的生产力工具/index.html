Emmet-强悍的编辑器插件

Posted by Rubicker on 2016-04-09

ST3中Emmet插件已经安装好久了，不过之前只熟悉其中的几个快捷键，今天学习了Emmet的官方文档，深感这个插件简直强大到没朋友，记录下来方便以后使用。

缩写

元素

直接输入元素的名字，之后Tab即可生成HTML标签

嵌套操作符

  • 子元素操作符 >
  • 兄弟元素操作符 +

  • ^ 返回上层，多个^连写将向上逐层返回; >用来加深层次

  • 乘法 在元素的后面输入加数字可一次性生成多个相同元素

  • ()分组 后续元素将与分组的第一个元素同级

属性操作符

  • ID和Class均采用与CSS相同语法来给元素添加属性

  • 自定义属性同样采用CSS相同语法[attr]

  • 编号 $

    将$放在元素名，属性名或属性值中可以进行排序

    多个连写的$可以生成带有前导0的编号

    使用@修饰符可以改变编号的方向（默认为升序或降序-）及起点

文本 {}

为元素添加文本

CSS缩写

厂商前缀

在属性前或者缩写前放一个连字符，Emmet会自动给这个属性添加厂商前缀；

当然在Emmet中，默认自动添加相应的厂商前缀；

功能

以下快捷键可能与某些软件冲突

匹配标签

从插入标签位置开始查找标签及标签内容，并选中它。多次调用时将向外扩展Ctrl+, 或向内收缩选择Ctrl+shift+0

跳转到配对标签

Ctrl+Alt+J

在HTML标签的开始与关闭标记内跳转

包裹缩写

Shift+Ctrl+G

展开缩写并将选中的内容放在最后生成的元素中

跳转编辑点

Ctrl+Alt+→或者Ctrl+Alt+←

适用于HTML代码块，跳转到尖括号外侧

与网易云音乐快捷键冲突 :)

选择

Ctrl+Alt+,或者Ctrl+Alt+.

选择HTML中重要的部分，如标签名，完整的属性和属性值

切换注释

Shift+Ctrl+/

注释或取消注释选中内容，对于 HTML 是整个标签内容(包括其后代标签)，对于 CSS 是一条规则或整个属性。

删除标签

Shift+Ctrl+；

更新图片尺寸

Ctrl+U

将插入符放在 <img> 标签内，然后调用此功能，就可以添加或更新 width 与 height 属性

对于 CSS，将插入符放在属性值 url() 内，然后调用此功能，就可以添加或更新 width 与 height 属性。

计算简单的数学表达式

Shift+Ctrl+Y

增减数字

选中要修改的数字

以1为单位增减: Ctrl+↑ 或 Ctrl+↓

以0.1为单位增减: Alt+↑ 或Alt+↓

以10为单位增减: Shift+Alt+↑ 或Shift+Alt+↓

与网易云音乐快捷键冲突 :)

重构CSS值

Shift+Ctrl+R

将插入符所在的CSS属性值复制给所有带厂商前缀的属性中
