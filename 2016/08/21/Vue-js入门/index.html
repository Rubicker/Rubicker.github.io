简单入门

开发环境搭建及热更新

  • 首先安装Vue命令行
    npm install -g vue-cli
  • 创建基于”webpack”模版的新项目”my-project”
    vue init webpack my-project
  • 安装依赖
    cd my-project
    npm install
  • 运行本地服务器
    npm run dev
    由于Vue支持热更新，因此我们对文件所做的更改会即时反映到页面当中
